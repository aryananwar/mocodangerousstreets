Set up any auxiliary functions

First, to make the data easier to work with we'll create a function to separate the date/time columns of each dataset into date and time columns. This will be easier for us to work with when trying to work on two sets of data as it will be in the same time period. Additionally, this will allow for greater control in specifying how accurate we want to get with time information in our analysis of the data set. We also added another function to standardize_time, for the sake of making sure that the data sets account for data gathered within the same year, we'll create a function to standardize the years of the data sets!

In[55]:
def define_date_and_time (df):
    
    date_column = df["Crash Date/Time"].map(lambda x: x.date())
    time_column = df["Crash Date/Time"].map(lambda x: x.time())
    temp_df= df.drop(["Crash Date/Time"], axis = 1)
    temp_df["date"]= date_column
    temp_df["time"]= time_column
    return temp_df

# Takes in a list of dataframs --> each of which should have a column named "date" which
# contains information about the year, month and day in a datetime object format
def standardize_time(dataframes:list):
    min_year=float('-inf')
    max_year=float('inf')
    for df in dataframes:
        min_year = max(min_year, df["Crash Date/Time"].min().year)
        max_year = min(max_year, df["Crash Date/Time"].max().year)
    return (min_year , max_year)

Logistic Regression
Model: Analysis, Hypothesis, Testing and Machine Learning






















During this phase of the datascience lifecycle, we plan to build hyptheses about which features of a road contribute most to the danger (in this case represented as fatality) for a pedestrian and make predicions based on these features and weights about which roads, during which times, surface conditions, and weather conditions are most likely to contribute to a dangerous scenario for a pedestrian.






















Firstly, we isolate the variables which make the most sense to contribute to pedestrian injury/fatality. (For example, leaving out the case identification number) and treat this as a matrix of feature vectors, treating every pedetrian traffic violation as an observation. For categorical values we use a One Hot Encoding to make the data easier to use for our regression analysis.


We then set the Fatality of each observation (a boolean describing whether an accident was fatal) as the true labels corresponding to each incident. Given that we  have a binary classification problem and have information regarding the ground truth of each observation, we opted for a Logistic Regression model. With our X and y set, we first apply logistic regression to our data set and observe the weights of each feature.

# Aryans attempt at logistic regression
pedestrian_df['time_in_minutes'] = pedestrian_df['time'].apply(lambda x: (x.hour*60)+ x.minute)
